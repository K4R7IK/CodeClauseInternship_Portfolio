Pixelify

This project is a full-fledged web application developed using a powerful tech stack that comprises React, Tailwind CSS, HTML, and JavaScript. Drawing inspiration from the popular platform Pinterest, the application offers users an intuitive and visually appealing space to share and discover captivating images.

The front-end, skillfully crafted with React, ensures a seamless and interactive user interface, elevating the overall user experience. The use of Tailwind CSS further enhances the aesthetics and responsiveness of the application, providing a polished and professional look.

For user authentication, the application seamlessly integrates with Google accounts, streamlining the sign-in process and bolstering security measures.

Behind the scenes, the project leverages Sanity as the backend, which plays a vital role in managing user accounts, facilitating image uploads, and securely storing relevant data. Sanity's flexible and scalable content infrastructure empowers the application to efficiently handle user-generated content while ensuring optimal performance.

Through this meticulous blend of front-end and backend technologies,the project delivers a captivating and user-centric image-sharing platform, embodying proficiency in modern web development practices.
